# Threads フックテンプレート集（AIニュース用）

データ根拠: @kudooo_ai 1,553件 + @asa_to_ame 1,154件の実測分析
生成日: 2026-03-10

---

## 使い方

1. 投稿1は **100字以内** に収める（100字以下: いいね平均15件）
2. スレッドは **3〜5件** で構成（5件: 単発の4.2倍のいいね）
3. パターンA〜Fを **ローテーション**（同じパターンを連続使用しない）
4. バズったフォーマットは **繰り返し再利用OK**

---

## パターンA: 速報型（表示平均3,638回）

AIニュースと最も相性が良い。タイムリーな公式発表・リリース情報に使う。

```
【速報】OpenAIが新モデルGPT-5を発表した。
これ、かなりデカい。
```

```
Gemini 2.5が来た。
Googleが本気出してきた。
```

```
Anthropicが新しい安全性フレームワークを発表。
業界のルールが変わる。
```

```
Meta、Llama 4をオープンソースで公開。
AI業界に激震。
```


## パターンC: やばい型（いいね平均136件）

```

```


## パターンE: 断言型

強い主張で注目を引く。トレンド変化・業界の転換点に使う。

```

これからはマルチモーダルAIの時代。
```

```

```


## パターンF: 質問型（返信誘発でアルゴリズム加速）

質問で返信を誘い、アルゴリズムを加速させる。

```

これマジで開発の仕方が変わる。
```

```

```

```

```

## スレッド構成テンプレート（3〜5件）

### 3件構成（標準）

```
投稿1（〜100字）: フック + URL
投稿2（150〜300字）: ①②③ リストで要点
投稿3（〜200字）: 「つまり〇〇ってこと」で締め
```

### 4件構成（詳細あり）

```
投稿1（〜100字）: フック + URL
投稿2（150〜300字）: ①②③ リストで要点
投稿3（100〜200字）: 深掘り・補足情報
投稿4（〜100字）: 「使ったことある？」等の質問で返信誘発
```

### 5件構成（最高効率）

```
投稿1（〜100字）: フック + URL
投稿2（150〜300字）: ①②③ リストで要点
投稿3（100〜200字）: 深掘り・技術的な補足
投稿4（100〜200字）: 実用的な活用例
投稿5（〜100字）: 質問 or まとめ
```

## NG集（データで証明済み）

| NG | 理由 |
|----|------|
| 200字以上のフック | いいね平均が1/4に低下 |
| 単発投稿 | 5件スレッドと比べていいね4.2倍低い |
| 「震えた」「やばいやつ来た」 | 禁止ワード |
| AI的比喩（「新たな地平」等） | 不自然 |
| 絵文字の多用 | @kudooo_aiは0%で成功。使うなら1〜2個まで |
