## 1. 投稿時間・曜日

### ベストな投稿時間

| 時間帯 | 効果 | 根拠（kudooo_ai実測データ） |
|--------|------|---------------------------|
| **夜18〜21時** | 最もインプが高い | 平均2,242imp。リラックスタイムでじっくり読まれる |
| **朝6〜9時** | 勝ち投稿が最も多い | 勝率3.5%、勝ち投稿8件。通勤時間にフックが刺さる |

### 絶対に避けるべき時間帯

| 時間帯 | 理由 | データ |
|--------|------|--------|
| **午後15〜18時** | 勝ち投稿ゼロ | 平均913imp。10,000impを超えた投稿が1つもない |
| **午前9〜12時** | 勝ち投稿ゼロ | 平均582imp。全時間帯で最低パフォーマンス |

> 同じ内容でも時間帯を変えるだけでインプが **3.8倍** 変わる（kudooo_ai）

### 曜日別の効果

| 曜日 | 効果 |
|------|------|
| **土曜日** | 最強（平均2,422imp）。特に土曜朝6時が最高の組み合わせ |
| 火〜水の夜 | エンゲージメント率が最も高い |
| **水曜日** | 最弱（平均1,084imp）。水曜の午後は最悪の組み合わせ |

- **朝6時**: 2本セット（予約投稿で前日夜にセット）

- 土日は朝の投稿を **8〜9時** にずらす（ユーザーの起床が遅い）

## 2. スレッド構成の最適解

### 推奨スレッド長

| **3件** | 基本形（最多） | kudooo_ai: 47.6%が3件、asa_to_ame: 36%が3件 |

| 単発1件 | いいね特化 | 短文フック1発で瞬間的ないいねを稼ぐ |

### 各投稿の役割と推奨文字数

```
投稿1（50字）: 短文フック — 注意を引くだけに集中

投稿3（467字）: 具体策 — ①②③のアクションリスト + 注意点
投稿4（50字）: CTA — 「最新ノウハウはこちら▼」+ リンク

```
投稿1（58字）: 短文フック — 実績数字 or 感情表現で引く
投稿2（94字）: 要点 — 結論 + 理由 + ①②③の箇条書き

## 3. フック（1行目）の書き方

### パターン別の効果ランキング

| 順位 | パターン | いいね平均 | 推奨頻度 | 例 |
|------|---------|-----------|---------|-----|
| 1 | **ニュース・速報型** | 26.4件 | 速報時に使う | 「新ルール出ましたね。Threads×note完全終了です。」 |
| 2 | **やばい・ガチ型** | 259.5件※ | 月1〜2回 | 「やばいやばい。ガチでやばい。」（asa_to_ame） |
| 3 | **あのー型** | 21.1件 | 週1〜2回 | 「あのー、Threads午後3時とか4時に投稿しない方が良いですよ。」 |
| 4 | **損訴求型** | 7.1件 | 標準的に使う | 「Threadsで1日3投稿以下の人、マジで損してます。」 |
| 5 | **Tips告知型** | 8件 | 教育投稿で使う | 「解説すると〜」「〜のポイントを教えます」 |

※ asa_to_ameの「やばい・ガチ型」はバズ投稿1件の影響で突出。使い方次第で爆発力あり。

### フックの鉄則

- **文字数は50字以内**（kudooo_aiの54%が50字以下。短いほど強い）
- **1行目で指を止めさせる**。どんなに良い内容でも読まれなければ意味がない
- **否定系フック**（「〜しない方がいい」「〜の人、損してます」）のリーチは平均2倍以上
- **バズったフォーマットは使い回してOK**（内容を変えて同じ型を再利用。2人とも実践）
- **同じパターンを連続で使わない**。ローテーションを意識する

### フックのNG

- 200字以上のフック（効果激減）
- 「おはようございます」から始める（asa_to_ame: 「1行目の挨拶は絶対NG」）
- 「震えた」「やばいやつ来た」等のAI臭い表現
- 絵文字の多用
- 「今日は〇〇について書きます」のような前置き

### asa_to_ameのフック3要素

1. **共感**: あるあるネタを入れる
2. **権威性**: 実績を提示する（「200万インプ達成した僕が〜」）
3. **緊急性**: 今だけの情報だと伝える

## 4. アルゴリズム攻略

### 2026年の最重要変化: Dear Algo（トピックベース判定）

- フォロワーベース → **トピックベース** へ完全移行（2026年2月発表）
- MetaのAIが「この人は何の専門家か」を判定する時代
- テーマがブレてるアカウントは「見たくない」指定される
- 対応できる人とできない人でインプに **2.4倍** の差

### アルゴリズムに評価される投稿

| 評価される | 評価されない |
|-----------|-------------|
| 滞在時間が長い（長文 or コメント展開型） | 中途半端な長さ（100-300字）の投稿 |
| ジャンル一貫性がある | テーマがバラバラ |
| コメント欄で会話が発生 | 外部リンクへの誘導（リーチ低下） |
| 新機能を使っている | note誘導（リーチ93%以上減少） |
| ツリー投稿（スレッド形式） | 単発の短文投稿 |

### 具体的なアクション

1. **発信ジャンルを1つに絞り切る**。全投稿の8割をその専門ジャンルに集中
2. **一貫性を最低3ヶ月維持**。短期間でジャンルを変えると評価リセット
3. **Meta公式アカウント+ブログを毎朝チェック**。新機能は24時間以内にテスト
4. **外部リンクは投稿に直貼りしない**。「プロフィールのリンクから」に誘導
5. **「続きはnoteで」は完全NG**。Threads内で価値提供を完結させる
6. **投稿後30〜90分のエンゲージメント**でアルゴリズム評価が決まる

### asa_to_ameの視点

- **ツリー投稿がアルゴリズムで高評価**。「続きを見るために詳細を開く」行為が加点対象
- **拡散投稿2割 : 教育投稿8割**のバランスが最適
- **毎日投稿 = 単純接触効果**。アルゴリズム以前にまず毎日顔を出す

## 5. エンゲージメント獲得法

### コメント展開型が最強

- コメント展開型: **平均1,867imp**（通常の中文投稿660impの **2.8倍**）
- メイン投稿は50字の短文フック → コメント欄で詳しく展開

### 3部構成の推奨フォーマット（kudooo_ai）

```
メイン投稿: 短文フック（50字以下）
コメント1: 体験談 + データ根拠（400-500字）
コメント2: 具体的アクション①②③（400-500字）
```

### いいねを増やすテクニック

- **短文は保存（いいね）されやすい**（asa_to_ame: 「一瞬で理解できる内容が強い」）
- **ニッチ特化**: Threads運用テーマは平均1,487imp、AI活用テーマは平均631imp（2.4倍の差）
- **投稿後1時間以内にコメント返信を徹底**（アルゴリズム評価に直結）
- **保存を促す文言を自然に入れる**

### asa_to_ameの企画型エンゲージメント

- **無料配布企画**: 「欲しい方は ①フォロー ②『○○』とコメント」→ いいね1,260件達成
- **AI画像シリーズ**: 「何枚目が好きですか？」でコメント誘導 → いいね2,329件
- **全コメントに返信**（短くてもOK。絵文字1つでもOK）

## 6. フォロワー獲得・成長戦略

### 共通の鉄則: ジャンル一貫性

**kudooo_aiもasa_to_ameも共通して最重要と言っているのが「ジャンルの一貫性」。**

- **発信ジャンルを1つに絞り切る**（「何の人か」が分からないとフォローされない）
- **全投稿の8割**を専門ジャンルに集中（残り2割は人柄投稿でOK）
- **一貫性は最低3ヶ月維持**。短期間で変えるとAI評価がリセットされる
- 「お昼ご飯は〇〇食べた」のような日常投稿は混ぜない

> 「この人は何の発信をしている人か」が一目でわからなければ、フォローにつながることはありませんよ。（asa_to_ame）

### 相互フォロー・フォロバは絶対NG（kudooo_ai）

- フォロバ企画で増えたフォロワーはテーマに興味がない → AIが「何のテーマか不明」と判定
- フォロワー数だけ増えて売上ゼロの最悪パターン
- 「数ではなく質」。LINE登録724件はフォロバ一切なしで達成

### asa_to_ameの成長実績

- 56日間でフォロワー **6,000→10,000人** 目前
- 1日で **フォロワー+1,000人** を達成したことあり
- 方法: **フック → ジャンル一貫 → 継続** の3ステップ

## 7. プロフィール・ブランディング

### プロフィール最適化3ステップ（kudooo_ai）

1. **1行目にジャンルキーワード**を入れる（アルゴリズムが拾えるワードを最初に）
2. **実績は数値で書く**（「フォロワー3,100名増」「LINE登録724件」。「実績多数」はNG）
3. **リンクは1つに絞る**（LINEかLPの1本に集約。5個超でエラー）

### 権威性の出し方（asa_to_ame）

- **「何者かである」ことを示せば信頼されて、そこからイージーモード**
- 実績が小さくても大きく見せる工夫をする
- 自信満々な言い切り口調を使う（「〜した者勝ちです」）
- 「〇〇のプロ」と名乗る（後発組が参入してくる前の今がチャンス）
- 投稿内で繰り返し実績を提示（「スレッズ上手すぎて」を定型フレーズ化）

> 遠慮なんていりません。使えるものは使いましょう。（asa_to_ame）

### 注意点

- プロフィール変更直後は一時的にリーチが落ちることがある（kudooo_ai）
- 「マーケター/発信者」→「Threads特化の運用データを発信」に変えたらフォロワー増加ペースが激変した

## 8. CTA・導線設計

### CTA配置の基本ルール

| ルール | kudooo_ai | asa_to_ame |
|--------|-----------|------------|
| CTA付きスレッドの割合 | **90.1%** | ほぼ全スレッドにリンク |
| CTA配置位置 | 最終投稿（短文CTA） | 投稿2〜3（リンク + 説明文） |
| 主なCTA先 | LINE、note、フォロー | ポスロボ、オプチャ、フォロー |

### 効果的なCTAパターン

**kudooo_ai型（フォロー + コンテンツ誘導）:**
- 「最新Threads運用の情報を見逃したくない方は、僕を（@kudooo_ai）フォローしておいてください。」

- 「欲しい方は ①このアカウントをフォロー ②『○○』とコメント」
- 段階的おねだり: フォロー → コメント → いいね → 再投稿

### 外部リンクの扱い

- **投稿にURLを直貼りするとリーチが下がる**（kudooo_ai）
- 「プロフィールのリンクから」に誘導する形が安全
- CTAはコメント欄の深い階層に配置
- **「続きはnoteで」は完全NG**（リーチ93%以上減少）

## 9. コンテンツ企画

### 拡散投稿 vs 教育投稿（asa_to_ame）

| 種類 | 比率 | 目的 | 例 |
|------|------|------|-----|
| **拡散投稿** | 2割 | 新規層の獲得 | 共感系、あるあるネタ、画像シリーズ |
| **教育投稿** | 8割 | ファン化・信頼構築 | ノウハウ、データ分析、具体策 |

> バズ狙いばかりだと濃いファンはつきません。信頼残高を貯める地味な投稿こそが資産になります。（asa_to_ame）

### 企画アイデア（asa_to_ame実績付き）

| 企画 | いいね実績 | ポイント |
|------|-----------|---------|
| **AI画像シリーズ** | 2,329件 | 「感性が似ている人に届く」フック + 「何枚目が好き？」コメント誘導 |
| **無料配布企画** | 1,260件 | フォロー + コメント条件でリード獲得 |
| **ツール無料化** | 219〜238件 | 段階的に配布。「正直やりすぎました」の自虐フック |

### コンテンツの鉄則

- **Threads内で価値提供を完結させる**。外部に飛ばさない（kudooo_ai）
- **出し惜しみしない**。全部書くとアルゴリズム評価が上がる
- **「新機能試してみた結果」系はバズりやすい**（kudooo_ai）
- **自分のデータや経験を必ず入れる**。テンプレ感のある投稿は不利
- **バズった型を使い回す**。構成やリズムを真似して内容だけ変えて再投稿（2人とも実践）

### データ引用で権威性を出す（kudooo_ai）

- 「N件のデータを分析した結果」（118回使用）
- 「Nヶ月でフォロワーN名達成」（217回使用）
- 「N万インプ」（154回使用）
- 「勝率N%」（具体的な数字で説得力）

### 共感構造（kudooo_ai）

```
僕も最初は〜してたんです。（共感）
でも〜したらマジで変わりました。（転換）
なぜかっていうと〜（理由）
じゃあ具体的にどうすればいいかっていうと▼（誘導）
```

### asa_to_ameの文体特徴

- **簡潔・言い切り**。「〜した者勝ちです」「〜してください」
- **F-PREP法**: フック → 結論 → 理由 → 具体例 → まとめ
- **一人称は「僕」**（83回）
- 「〜です。」の断定語尾（294回）が中心

### 避けるべき表現

- 「〜です。〜ます。」の単調な繰り返し（AI臭）
- 「重要です」「必要です」などの抽象ワード
- 体験談がない一般論のみの投稿
- 語尾が3回連続で同じになること（離脱率が上がる）

## 12. マインドセット・継続

### 2人の共通メッセージ

**「継続した人だけが勝てるゲーム」**

- kudooo_ai: 完璧な1本より **6割の完成度で5本** 出す
- asa_to_ame: **毎日5投稿** を息をするように続ける（最低ライン）

### 具体的なアクション

| ルール | 詳細 |
|--------|------|
| **量 > 質** | 量をこなした先に質がついてくる。投稿を量産するとどのフックが伸びるか分かるようになる |
| **検証マインド** | 「上手く書く」のではなく「違いを作って反応の差を見る」。毎回1か所だけ変えてA/Bテスト |
| **低パフォーマンス投稿は削除** | 24時間後にエンゲージメントが低い投稿は削除（アカウント評価を下げないため） |
| **自動化で継続のハードルを下げる** | 毎日投稿を仕組み化する（asa_to_ame: 実際に56日で1,154件 = 1日平均20投稿） |
| **焦らない** | ただし止めないことが最重要 |

> 大切なのは発信じゃなく検証。何が刺さって何が無視されたかを見ないから一生伸びないまま。（asa_to_ame）

> 投稿して反応見て改善するサイクルが最強。アルゴリズムの変化を恐れすぎて投稿を止めないこと。（kudooo_ai）
