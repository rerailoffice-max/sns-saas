## @kudooo_ai の分析（1,553件・確定版）

### 投稿スタイル
教育・ノウハウ系。スレッド形式（3〜4投稿が多い）で価値提供し、最後にCTA（LINE登録・特典配布）を入れる構成。「あのー、〜しない方が良いですよ」という問題提起型フックがいいね最多。絵文字・タイトル行はほぼ使わず口語フックで始める。

### 冒頭パターン（投稿1の書き出し・確定版）

| パターン | 例 | 特徴 |
|---------|---|------|
| 問題提起・損訴求型 | `あのー、Threads午後3時とか4時に投稿しない方が良いですよ。` | いいね最多（408件）。親しみやすい「あのー」で始める |
| 実績訴求型 | `5ヶ月でフォロワー4342名、455万インプ達成した僕が〜` | 数字を先出しして信頼を獲得 |
| 業界ニュース反応型 | `新ルール出ましたね。Threads×note完全終了です。` | タイムリーな話題でいいね184件 |
| Tips告知型 | `Threadsのアルゴリズム変わりました。2026年の変更点をお伝えすると` | 表示回数7,966回（最高水準） |
| NotebookLM・ツール活用型 | `Threads投稿をNotebookLMで分析してない人、損してます。` | 表示回数8,019回（最高） |

### スレッド構成パターン

```
投稿1（347字平均）: フック（問題提起 or 実績数字 or ニュース反応）
                   ↳ URLリンクがある場合はここに配置（13%）
投稿2（243字平均）: ノウハウ本体（①②③番号リスト）
投稿3（243字平均）: 深掘り or まとめ
投稿4（任意）: 追加詳細 or CTA（LINE登録リンク等）
```

### 文体の特徴（1,553件・確定版）

- **口語・親密感重視**: 「あのー、」「マジで」「めっちゃ」「〜んです」
- **改行が多い**: 1文ごとに改行。モバイルで読みやすいフォーマット
- **タイトル行はほぼ不使用**: 1,553件中21件（1%）のみ。`【】`囲みより口語フックが主流
- **数字を必ず入れる**: `4342名` `455万インプ` `300文字` `5ヶ月` など具体的な数字
- **絵文字なし**: 1,553件中3件（0%）。▼ ✅ 等の記号は使用
- **リンク位置**: 投稿1に含める場合（13%）と、CTA専用の最終投稿に置く場合の2パターン
- **敬語なし**: タメ口〜やや丁寧語（「〜ます」は使うが基本タメ口）
- **①②③リスト**: 2投稿目以降に番号付きリストで具体的ノウハウを展開

### バズった投稿パターン（いいね・表示回数上位）

**いいね上位（エンゲージメント型）**
> あのー、Threads午後3時とか4時に投稿しない方が良いですよ。（いいね408件）

> 新ルール出ましたね。Threads×note完全終了です。（いいね184件）

> あのー、Threadsは午後に投稿しない方がいいですよ。（いいね171件）

→ 「あのー、〜しない方が良い」という逆張り・禁止系フックが最も反応が高い

**表示回数上位（リーチ型）**
> Threads投稿をNotebookLMで分析してない人、損してます。（表示8,019回）

> Threadsのアルゴリズム変わりました。2026年の変更点をお伝えすると（表示7,966回）

→ ツール活用Tips・アルゴリズム変化の業界動向が表示回数最大

## @asa_to_ame の分析（1,154件・確定版）

### 投稿スタイル
「実績者＋ツール提供者」ポジション。短いフック（78字平均）＋AI画像＋無料配布でバズを作り、自作ツール『ポスロボ』へ誘導する構成。同じフレーズの繰り返し戦略が特徴的。スレッド平均2.7件で、5〜7件スレッドが最高パフォーマンス。

### 冒頭パターン（投稿1の書き出し・確定版）

| パターン | 例 | いいね平均 |
|---------|---|---------|
| 感性型（画像シリーズ） | `Threadsは感性が似ている人に届くらしい。何枚目が好き？` | **560件** |
| やばい・ガチ型 | `やばいやばい。ガチでやばい。Threads×AI界隈に激震走ります。` | **136件** |
| 無料配布型 | `【Threads自動投稿ツールを完全無料で配布】します。↓` | 55件 |
| 速報型 | `【速報】ワイ、Threads運用のノウハウ喋りすぎな件` | 48件 |
| 実績自慢型 | `スレッズ上手すぎて、毎日フォロワーが爆増してる。` | 39件 |

### スレッド構成パターン

```
投稿1（78字平均）: 短いフック（実績数字 or 感情爆発 or AI画像）
投稿2（108字平均）: ノウハウ or リプ返信
投稿3（〜221字）: 🔗 ポスロボリンク + オプチャ誘導
投稿4以降: リプ返信でコミュニティ形成
```

### 文体の特徴（1,154件・確定版）

- **簡潔・超短文**: 1投稿あたり平均78字（@kudooo_aiの347字の1/4以下）
- **敬語ベース**: 「〜です」「〜ます」が49%
- **感情表現を混ぜる**: 「やばい」「ガチで」「めちゃくちゃ」が9%
- **絵文字あり**: 16%（🫶✌️🔥 等）。@kudooo_aiの0%とは対照的
- **画像活用**: 12%がAI生成画像付き（画像がバズの起爆剤）
- **リンク位置**: 投稿2〜3にポスロボリンクを配置
- **同じフレーズの繰り返し**: 「スレッズ上手すぎて」を10回以上再利用
- **質問で返信誘発**: 「何枚目が好き？」「教えてください」が5%

## AI情報ニュース投稿への適用ポイント（2アカウント分析確定版）

### 2アカウント比較から導いた最適戦略

| 要素 | 採用元 | 戦術 |
|------|--------|------|
| フック文体 | @kudooo_ai | 「あのー、〇〇しない方がいいですよ」（週1〜2回） |
| フック長 | @asa_to_ame | 1投稿目は100〜200字（短いフックが強い） |
| スレッド長 | @asa_to_ame | 3〜5件が最適バランス（5〜7件で最高効率） |
| 画像 | @asa_to_ame | AI生成画像を毎回添付（12%→常時に強化） |
| テーマ | @kudooo_ai | AI業界ニュース・ツール活用Tips |
| 繰り返し | @asa_to_ame | バズったフォーマットは繰り返し使う |
| CTA | 両者参考 | 投稿2〜3でリンク配置（自然な導線） |
| 投稿曜日 | 両者参考 | 火曜（表示最強）・水曜（いいね最強）を強化 |

### 特に使えるパターン（重要度順）

1. **逆張り・問題提起型フック**: 「あのー、まだ〇〇してない人は損してます」
   → @kudooo_aiでいいね平均21.8件（最強）
   → 例: `あのー、まだGemini 2.0を使ってない人、マジで損してます。`

2. **速報型フック**: 「【速報】〇〇」「〇〇が変わりました」
   → @asa_to_ameでいいね平均48.3件、表示平均3,638回。AIニュースと最も相性が良い
   → 例: `【速報】OpenAIの料金体系が変わりました。`

3. **画像＋質問**: AI画像添付 +「どう思う？」「使ったことある？」
   → @asa_to_ameで560件平均。返信を誘発→アルゴリズム加速
   → 例: AI生成インフォグラフィック + `このツール使ったことある？`

4. **ツール活用Tips型**: 「〇〇を使ってない人、損してます」
   → @kudooo_aiで表示8,019回。@asa_to_ameでも高リーチ
   → 例: `Claude Codeを使ってない人、損してます。`

5. **バズったフォーマットの再利用**: 同じ構文を数字だけ更新して繰り返し投稿
   → @asa_to_ameの10回以上の再利用がペナルティなしで成立

6. **スレッドは3〜5件**: @kudooo_ai型の単発フック + @asa_to_ame型のスレッド展開を融合

### AI情報Bot特有の考慮事項
- @kudooo_aiは「ノウハウ教育」、@asa_to_ameは「ツール提供」。AI情報Botは「ニュース速報」
- **速報型フックが最も相性が良い**（@asa_to_ameで表示平均3,638回）
- 画像は@kudooo_ai 3%、@asa_to_ame 12%だが、AI情報Botは毎回画像ありが差別化になる
- 「あのー、」+「【速報】」の使い分けでフックにバリエーションを持たせる
- 無料配布型は使えないが、「無料で使える〇〇」としてツール紹介に応用可能
